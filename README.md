# TRY-ANGLE
体重・食事・トレーニングといった３つの観点からトレーニーをサポートできるアプリケーションです。

# アプリケーションの概要
・体重の管理          →  入力データを折線グラフ化し、体重推移を確認ができる
・食事の管理          →  入力データを一覧表示し、過去の食事を振り返る事ができる
・トレーニングの管理  →  入力データを一覧表示し、過去のトレーニングを振り返る事ができる
